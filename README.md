[js-mapping](http://aureooms.github.io/js-mapping)
==

Mapping library for JavaScript. Parent is
[js-library](https://github.com/aureooms/js-library).

```js
mapping.fromkeys( 'ab' , 1 ) ; // [['a', 1], ['b', 1]]
```
